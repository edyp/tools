# Tools
Different tools that I've created for simplifying my daily basis.

**count_statistics.py**
\nCount how many times test case pass and print statistics.
**remove10.py**
\nRemoves 10 first characters from every line in a file.

**telnet-dev**
\nExpect script for telneting and preparing a device to working/developing on it.
- telnet to device
- mount nfs
- execute something if needed

**logs_downloader.sh**
\nDownload logs from jenkins.
